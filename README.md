## Building (linux/wsl)

- Clone the repo
```bash
git clone https://github.com/nishantHolla/snake-asm.git
cd snake-asm
```

- Create the docker container
```bash
./docker.sh build
```

- Run the docker container
```bash
./docker.sh run
```
